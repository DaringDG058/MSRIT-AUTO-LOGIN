# MSRIT-AUTO-LOGIN
A Browser Extension to auto login to MSRIT's Attendance portal with your USN and DOB whenever you just visit the portal.

This Extension works on Chrome based Browsers only.

1. First download the two files "content.js" and "manifest.json" to any of the fresh folders on your computer. 
2. Open the "content.js" file with any code editor and enter your details there, save and close the file.
3. Open any of your CHROME based browsers and with the help of the three lined menu - go to the "EXTENSIONS" settings and "Manage Extensions". 
4. Enable "Developer Mode" in the top right corner and select the "Load Unpacked" option and select the folder in which you had saved the files "content.js" and "manifest.json" and hit select.
5. You are almost done, now just open the website = "https://parents.msrit.edu/newparents/" and see the Extension do the work for you!.
6. Add a bookmark of the Website for Faster Access.
